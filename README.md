# Employee Management System - Backend

A complete REST API backend built with **Spring Boot 3**, **JPA/Hibernate**, **MySQL**, and **Lombok**.

## Features
- Employee & Department CRUD operations
- Search employees by name, department, role
- Pagination and sorting
- DTO pattern
- Global exception handling
- Tested with IntelliJ HTTP Client

## Tech Stack
- Java 17
- Spring Boot 3
- Spring Data JPA
- MySQL
- Lombok

## How to Run
1. Clone the repo
2. Set up MySQL database `employee_db`
3. Update `application.properties` with your DB credentials
4. Run `EmsApplication.java`
5. Test APIs using `api-tests.http`
